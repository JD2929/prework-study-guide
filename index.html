  • CSS is the code the makes the webpage actually look good!

  • A padding indicates how much space we want around the content inside an element.
  • You must link the style css page to the code or it won't be connected
  • You can list several attributes at once if you want them to all look the same.
  • There are different ways to tell it a colour, by name, RGB, or hex code.
